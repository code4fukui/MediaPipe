# MediaPipe


MediaPipeは、マルチモーダルな応用MLパイプラインを構築するためのクロスプラットフォームフレームワークです。

## 特徴

- ハンドトラッキングを含む、ライブおよびストリーミングメディア向けのカスタマイズ可能な機械学習ソリューションのセットを提供します。
